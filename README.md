<h2> 👨🏻‍💻 &nbsp;About Me </h2>

- 👨🏻‍💻 &nbsp; Meu primeiro contato com programação aconteceu aos 17 anos, quando decidi criar um servidor privado e uma página na internet. Depois de muita pesquisa e esforço, consegui finalmente colocar o servidor online. A sensação foi mágica, e esse momento ficou marcado na minha memória. Descobri que podia personalizá-lo do jeito que quisesse, apenas escrevendo linhas de código — algo que me fascinou profundamente. Foi então que percebi que era isso que eu queria fazer para o resto da minha vida: ser programador.
- 🚀 &nbsp; Atualmente, tenho como foco principal da minha carreira as áreas de desenvolvimento web e mobile. Utilizo a stack JavaScript (Node, React & React Native, Nestjs) para desenvolver em ambas as frentes.
- 🏎 &nbsp; Sou apaixonado por musica, especialmente por tipos Classicos, e costumo jogar simuladores no meu tempo livre. É o que eu mais gosto de fazer pra me divertir e esvaziar a cabeça haha!
- 💵 &nbsp; Também gosto muito de investimentos, do mercado financeiro e de economia.
